# Personal Assistant Skills

A collection of reusable [Claude Code](https://claude.ai/code) skills.

## Using a Skill

1. Browse the skills below
2. Copy the skill's directory into your project's `.claude/skills/` directory (or `~/.claude/skills/` for global access)
3. Restart Claude Code
4. Invoke the skill with `/<skill-name>`

## Available Skills

| Skill | Description |
|-------|-------------|
| [interview-prep](./interview-prep/) | Analyze a resume against a job description and generate a comprehensive interview prep guide |
